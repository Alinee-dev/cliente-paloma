# Cliente Paloma

Proyecto estático sencillo que contiene una página web con HTML, CSS y JavaScript. Este repositorio sirve como base para un sitio de cliente (landing, demo o página informativa).

**Estado:** plantilla funcional (HTML estático)

## Contenido
- **`index.html`**: Entrada principal del sitio.
- **`css/styles.css`**: Estilos y diseño.
- **`js/main.js`**: Lógica JavaScript del cliente.

## Requisitos
- Navegador moderno (Chrome, Edge, Firefox, Safari).

## Estructura del proyecto

```
index.html
css/
	└─ styles.css
js/
	└─ main.js
README.md
```

## Desarrollo
- Edita `css/styles.css` para cambiar estilos y diseño.
- Edita `js/main.js` para añadir o modificar comportamiento del cliente.
- Si agregas imágenes o assets, crea una carpeta `assets/` o `img/` en la raíz.

Buenas prácticas:
- Mantén el CSS modular y usa clases semánticas.
- Evita dependencias innecesarias en JavaScript si el sitio es estático.
